Trabajo final de la asignatura Introducción a las Ciencias de la Computación y Programación en la universidad nacional de Colombia semestre 2021-2
Trabajo fue realizado por Augusto David Rico Dautt y Nicole Sofía Salazar Delgado
El trabajo consiste de un sudoku que se autogenera y que es válido, ya que se confirma que tiene única respuesta y utilizando la librería pygames se genera un entorno gráfico para rellenarlo
al momento de llenarlo correctamente se muestra un mensaje de felicitaciones y se puede guardar la puntuación.
Las mayores puntuaciones pueden ser consultadas en el menú de inicio.

Recomendaciones: el programa fue testeado en linux mint 20.3 usando python 3.9.5 por lo que se recomienda emplear el mismo para su desarrollo o carga, no obstante debe poder ejecutarse en Windows de igual manera.

Requisitos: debe tenerse instalada la librería 'Pygame' para poder ejecutar el programa, de esta librería se empleó la versión 2.1.2 por lo que Se recomienda emplear la misma.
Para instalar la librería 'Pygame' se recomienda utilizar el comando 'pip install -r requirements.txt' estando en la carpeta para instalar fácilmente la versión 2.1.2
